# Code Frame Assignment App

Features :
- Add Actions
- Delete Actions
- History Logs

## Running the app locally

### Clone the repository

```bash
git clone https://github.com/siddastic/code_frame.git
```

### Install dependencies

Run in project root directory

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```
